# Timer Stack

A beautiful, functional timer and stopwatch application built with React, Vite, and Tailwind CSS.

## Features

- **Timer**: Set countdowns with custom times.
- **Stopwatch**: Track elapsed time with precision.
- **Modern UI**: Clean and responsive design using Tailwind CSS and Framer Motion.

## Run Locally

**Prerequisites:** Node.js

1. Clone the repository:
   ```bash
   git clone https://github.com/website-deployer/timer.git
   cd timer
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the app:
   ```bash
   npm run dev
   ```

## Technologies Used

- React 19
- Vite
- Tailwind CSS 4
- Framer Motion
- Lucide React
